# AdjMatrixGraph (CSCE146)

A Java implementation of a graph represented by an adjacency matrix, featuring both depth-first and breadth-first search algorithms.

## Files
- `AdjMatrixGraph.java` — An adjacency matrix–based implementation featuring DFS and BFS methods.
- `AdjMatrixChecker.java` — A simple test runner containing the main method to demonstrate the graph in action.

## How to compile & run (command line)
1. Clone repo or download files.
2. Compile:
   ```bash
   javac AdjMatrixGraph.java AdjMatrixChecker.java
3. Run:
java AdjMatrixChecker
